# Cinema System - Информационная система кинотеатра

Курсовой проект по дисциплине "[Название дисциплины]"

## Описание
Система управления кинотеатром с функциями бронирования билетов, управления сеансами и клиентами.

## Функциональность
- Управление фильмами, залами и сеансами
- Бронирование билетов с выбором мест
- Система лояльности для клиентов
- Поиск и фильтрация сеансов

## Технологии
- C++17
- Qt6
- CMake

## Сборка
```bash
mkdir build && cd build
cmake ..
cmake --build .
